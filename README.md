# PRY_MI_PRIMERA_PAGINA_DINAMICA

## Descripción
Este proyecto consiste en la personalización de un templade web con el objetivo de adaptarlo a nuestras necesidades y aprender el uso práctico de HTML, CSS, JavaScript y Bootstrap.  
El proyecto demuestra cómo modificar una plantilla existente para crear una página dinámica y visualmente atractiva que se ejecuta directamente en el navegador.

## Objetivo
Aplicar los conocimientos adquiridos en programación web para comprender la estructura, el diseño responsivo y la interacción básica entre las tecnologías del lado del cliente.

## Tecnologías utilizadas
- HTML5 → Estructura base del sitio.  
- CSS → Estilos y diseño visual.  
- JavaScript → Dinamismo e interactividad.  
- Bootstrap 5 → Sistema de diseño responsivo y componentes predefinidos.  

## Ejecución del proyecto
1. Descarga o clona el repositorio:
   ```bash
   git clone https://github.com/usuario/PRY_MI_PRIMERA_PAGINA_DINAMICO.git

2. Abre la carpeta del proyecto.

3. Ejecuta el archivo index.html directamente en tu navegador (doble clic o arrastrar el archivo al navegador).
